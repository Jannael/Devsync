## Resumen

`DEVSYNC.json` es la única fuente de verdad para tu portafolio, CV, perfil de GitHub, LinkedIn y académicos. Toda tu información se define aquí y se sincroniza en todas las plataformas.

<Aside type="tip">
Puedes omitir campos que no necesites. La plantilla simplemente no los mostrará. Sin embargo, la capacidad de generar salidas depende de la plantilla que estés usando.
</Aside>

## Campos Globales

Estos campos aplican a todos los idiomas y salidas:

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `defaultLang` | `string` | Sí | Código de idioma predeterminado (ej. `"en"`, `"es"`, `"fr"`) |
| `site` | `string` | Sí | URL de tu sitio web del portafolio |
| `name` | `string` | Sí | Tu nombre completo |
| `img` | `string` | Sí | URL de la imagen de perfil |
| `socialMedia` | `array` | No | Array de perfiles de redes sociales |
| `githubUserName` | `string` | Sí | Tu nombre de usuario de GitHub |
| `address` | `string` | Sí | Tu ubicación (ciudad, país) |
| `email` | `string` | Sí | Tu dirección de correo electrónico |
| `phone` | `string` | Sí | Tu número de teléfono |

### Ejemplo: Campos Globales

```json
{
  "defaultLang": "en",
  "site": "https://devsync.work",
  "name": "Tu Nombre Completo",
  "img": "https://your-image-url.com/your-image.jpg",
  "socialMedia": [
    {
      "name": "LinkedIn",
      "url": "https://www.linkedin.com/in/your-linkedin-handle",
      "icon": "/icons/linkedin.svg",
      "mdBadge": "![LinkedIn](https://img.shields.io/badge/LinkedIn-0A66C2?logo=linkedin&logoColor=white)"
    }
  ],
  "githubUserName": "tu-usuario-github",
  "address": "Tu Ciudad, País",
  "email": "tu-email@example.com",
  "phone": "+1 123 456 7890"
}
```

## Campos Específicos por Idioma

Todos los campos bajo códigos de idioma (ej. `en`, `es`, `fr`) se traducen por idioma.

### Estructura

```json
{
  "es": {
    "description": "Resumen profesional",
    "jobTitle": "Tu Título de Trabajo",
    "status": { ... },
    "languages": [ ... ],
    "experience": [ ... ],
    "projects": [ ... ],
    "education": [ ... ],
    "certifications": [ ... ]
  }
}
```

### Referencia de Campos

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `description` | `string` | Sí | Resumen profesional/biografía |
| `jobTitle` | `string` | Sí | Título de trabajo actual |
| `status` | `object` | No | Estado de disponibilidad actual con badge |
| `languages` | `array` | No | Idiomas hablados |
| `experience` | `array` | No | Experiencias laborales |
| `projects` | `array` | No | Proyectos |
| `education` | `array` | No | Educación |
| `certifications` | `array` | No | Certificaciones |

## Objeto de Estado

```json
{
  "status": "Abierto a ofertas",
  "badge": "![Estado](https://img.shields.io/badge/Estado-Abierto%20a%20ofertas-green)"
}
```

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `status` | `string` | Texto de estado (ej. "Abierto a ofertas", "Empleado") |
| `badge` | `string` | Badge markdown para el estado |

## Entrada de Experiencia

```json
{
  "company": "Nombre de la Empresa",
  "position": "Tu Puesto",
  "img": "https://company-logo.com/logo.jpg",
  "web": "https://company-website.com",
  "description": "Una oración describiendo tu rol",
  "date": "MM/YY - Presente",
  "links": [ ... ],
  "list": {
    "title": "logros-clave",
    "items": [ ... ]
  },
  "skills": [ ... ]
}
```

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `company` | `string` | Sí | Nombre de la empresa |
| `position` | `string` | Sí | Tu título de trabajo |
| `img` | `string` | No | URL del logo de la empresa |
| `web` | `string` | No | Sitio web de la empresa |
| `description` | `string` | Sí | Descripción del rol |
| `date` | `string` | Sí | Rango de fechas (ej. "01/23 - Presente") |
| `links` | `array` | No | Enlaces adicionales |
| `list.title` | `string` | No | Título de la sección |
| `list.items` | `array` | No | Elementos de logros |
| `skills` | `array` | No | Habilidades usadas |

### Elemento de Logro

```json
{
  "highlight": "Título del logro",
  "description": "Describe un resultado medible que lograste"
}
```

## Entrada de Proyecto

```json
{
  "name": "Nombre del Proyecto",
  "img": "https://project-image.com.jpg",
  "web": "https://project-url.com",
  "links": [ ... ],
  "description": "Lo que hace el proyecto y por qué importa",
  "list": {
    "title": "aspectos-destacados",
    "items": [ ... ]
  },
  "skills": [ ... ]
}
```

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `name` | `string` | Sí | Nombre del proyecto |
| `img` | `string` | No | URL de imagen del proyecto |
| `web` | `string` | No | Sitio web del proyecto |
| `links` | `array` | No | Enlaces relacionados (GitHub, demo, etc.) |
| `description` | `string` | Sí | Descripción del proyecto |
| `list` | `object` | No | Sección de aspectos destacados |
| `skills` | `array` | No | Habilidades/tecnologías usadas |

## Validación

La configuración se valida usando Zod. Si la validación falla, verás un error como:

```
Estructura de DEVSYNC.json inválida
Campo de perfil requerida faltante
```

Verifica que todos los campos requeridos estén presentes y correctamente formateados.
