---
title: Traducciones
description: Idiomas disponibles y sus códigos de locale.
---

Devsync soporta los siguientes idiomas para las etiquetas de la interfaz en las salidas generadas.

| Código | Idioma    | Locale  |
| ------ | --------- | ------- |
| `en`   | English   | `en_US` |
| `es`   | Español   | `es_ES` |
| `fr`   | Français  | `fr_FR` |
| `pt`   | Português | `pt_PT` |
| `de`   | Deutsch   | `de_DE` |
| `zh`   | 中文      | `zh_CN` |
| `ja`   | 日本語    | `ja_JP` |
| `ko`   | 한국어    | `ko_KR` |
