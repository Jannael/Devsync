## Problemas de Instalación

### Instalación de Puppeteer Falla (Local)

**Error:** `Error: Cannot find module 'puppeteer'` o la instalación se cuelga

**Causa:** Dependencias del sistema faltantes o problemas de red

**Solución:**

```bash
# Instalar dependencias de Chrome (Linux/WSL)
sudo apt-get update
sudo apt-get install -y \
  libx11-6 libxcomposite1 libxcursor1 libxdamage1 libxext6 \
  libxfixes3 libxi6 libxtst6 libnss3 libglib2.0-0 libgtk-3-0t64 \
  libasound2t64 libxrandr2 libpangocairo-1.0-0 libatk1.0-0 \
  libatk-bridge2.0-0 libcups2 libdbus-1-3 libdrm2 libgbm1 \
  libgdk-pixbuf2.0-0
```

<Aside type="note">
	En macOS, Puppeteer generalmente funciona sin configuración adicional. En Windows, asegúrate de tener lo último en Visual C++ Redistributable
	instalado.
</Aside>

### Bun No Encontrado (Local)

**Error:** `bash: bun: command not found`

**Solución:**

```bash
# Instalar Bun
curl -fsSL https://bun.sh/install | bash

# O con npm
npm install -g bun
```

Verifica la instalación:

```bash
bun --version
```

## Problemas del Comando Build (Local)

### Chrome No Encontrado (Generación de PDF)

**Error:** `Failed to launch the browser process` o `Chrome not found`

**Causa:** Puppeteer no puede encontrar Chrome/Chromium

**Solución:**

1. Instala las dependencias de Chrome (ver sección Puppeteer arriba)
2. Establece Puppeteer skip download en `.env`:

```bash
PUPPETEER_SKIP_DOWNLOAD=true
```

3. Asegúrate de que Chrome/Chromium del sistema esté instalado:

```bash
# Ubuntu/Debian
sudo apt-get install chromium-browser

# macOS
brew install --cask google-chrome
```

### Errores de Validación de DEVSYNC.json

**Error:** `Invalid DEVSYNC.json structure` o `Missing required profile field`

**Causa:** Validación de esquema falló

**Solución:**

1. Verifica que todos los campos requeridos estén presentes:
   - `defaultLang`
   - `site`
   - `name`
   - `img`
   - `githubUserName`

2. Verifica la sintaxis JSON (sin comas finales, comillas apropiadas)

3. Verifica que los tipos de campo coincidan con los valores esperados

Ejemplo inválido vs válido:

```json
// ❌ Inválido - faltan comillas
{
  name: John Doe, // Debería ser "name": "John Doe"
  "site": https://example.com // Debería ser "https://example.com"
}

// ✅ Válido
{
  "name": "John Doe",
  "site": "https://example.com"
}
```

### PNPM create-template no funciona

`pnpm` tiene reglas de seguridad estrictas para ejecución de scripts que podrían prevenir que `create-template` funcione sin configuración adicional. Recomiendo usar `bunx` en su lugar. Como `bunx` solo usa características incorporadas de Bun para copiar la carpeta `@core`, no hay problemas de seguridad.

## Problemas de Git

### Permiso Denegado al Subir

**Error:** `remote: Permission denied` o `fatal: could not read Username`

**Causa:** Credenciales de GitHub faltantes o incorrectas

**Solución:**

Para GitHub Actions:

```yaml
permissions:
  contents: write
```

Para Git local:

```bash
# Verificar remote URL
git remote -v

# Re-autenticar
git credential-cache exit
git push
```

## Problemas de Plantillas

### Plantilla No Encontrada

**Error:** `Template DEVSYNC.json not found` o `Failed to clone repository`

**Causa:** URL de plantilla inválida o el repositorio no existe

**Solución:**

1. Verifica que la URL de la plantilla sea correcta:

```bash
# Debería ser: https://github.com/[owner]/[repo]
bunx @jannael/devsync init --template @jannael/my-template
```

2. Verifica que el repositorio exista y sea público

3. Usa la plantilla predeterminada si no estás seguro:

```bash
bunx @jannael/devsync init
```

## Problemas de Generación de PDF

### PDF No Generado

**Error:** Falta archivo PDF en la salida

**Causa:** Conversión HTML a PDF falló

**Solución:**

1. Verifica que `package.json` tenga la ruta correcta:

```json
{
	"devsync": {
		"pathToCompiledCV": "./dist/[lang]/cv/index.html"
	}
}
```

2. Verifica que el archivo HTML exista en esa ruta

3. Verifica que el comando build realmente genere el HTML

### Problemas de Calidad de PDF

**Problema:** El PDF se ve cortado o mal formateado

**Solución:**

1. Asegúrate de que el HTML tenga la etiqueta meta viewport apropiada
2. Verifica que el CSS no tenga anchos fijos que excedan el tamaño de página
3. Prueba la generación de PDF localmente primero

## Problemas de GitHub Actions

### El Flujo No Se Activa

**Problema:** Push a `DEVSYNC.json` pero el flujo no se ejecuta

**Solución:**

1. Revisa la pestaña Actions - el flujo podría estar deshabilitado
2. Verifica que `DEVSYNC.json` esté en la raíz (no en subdirectorio)
3. Verifica la ruta del archivo de flujo: `.github/workflows/*.yml`
4. Verifica la sintaxis YAML:

```yaml
on:
  push:
    paths:
      - 'DEVSYNC.json' # Coincidencia exacta requerida
```

### El Trabajo Falla en Ubuntu

**Error:** Varios errores en el runner `ubuntu-latest`

**Solución:**

Agrega todas las dependencias requeridas:

```yaml
- name: Install dependencies
  run: |
    sudo apt-get update
    sudo apt-get install -y libx11-6 libnss3 libgtk-3-0t64 libxcomposite1 libxcursor1 libxdamage1 libxext6 libxfixes3 libxi6 libxtst6 libxrandr2 libpangocairo-1.0-0 libatk1.0-0 libatk-bridge2.0-0 libcups2 libdbus-1-3 libdrm2 libgbm1 libgdk-pixbuf2.0-0 libasound2t64
```

## Problemas de Rendimiento

### Tiempos de Build Lentos

**Problema:** Build toma >5 minutos

**Solución:**

1. Dependencias en caché:

```yaml
- name: Cache Bun dependencies
  uses: actions/cache@v4
  with:
    path: ~/.bun/install/cache
    key: ${{ runner.os }}-bun-${{ hashFiles('**/bun.lockb') }}
```

2. Usa una versión específica de Bun en lugar de `latest`

3. Reduce tamaños de imagen en `DEVSYNC.json`

## Mensajes de Error Comunes

| Error                               | Causa                           | Solución                        |
| ----------------------------------- | ------------------------------- | ------------------------------- |
| `ENOENT: no such file or directory` | Ruta de archivo no existe       | Verifica rutas en config        |
| `EACCES: permission denied`         | Problema de permisos de archivo | Verifica permisos de archivo    |
| `ETIMEDOUT`                         | Tiempo de red agotado           | Verifica red, reintenta         |
| `Invalid URL`                       | URL malformada en config        | Corrige formato de URL          |
| `ZodError`                          | Validación falló                | Verifica tipos/valores de campo |
