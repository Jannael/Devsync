## Comando Build

El comando `build` genera todos los artefactos de tu portafolio desde `DEVSYNC.json`.

<Aside type="note">
	Este comando está pensado principalmente para GitHub Actions, pero puedes ejecutarlo localmente si tienes Bun y Puppeteer instalados.
</Aside>

<Steps>

1. **Instalar dependencias** usando `bun install`.

2. **Construir el proyecto** usando `bun run build`.

3. **Validar `DEVSYNC.json`** para asegurar que la configuración sea correcta.

4. **Construir CV para cada idioma** - Lee la configuración `devsync` para determinar la ruta al CV HTML compilado.

5. **Crear perfil de LinkedIn** para cada idioma configurado.

6. **Crear perfil de GitHub** usando el `defaultLang`. El estilo se controla con `devsync.githubProfileStyle` en `package.json` — `"default"` (perfil completo) o `"minimal"` (compacto).

7. **Crear perfil académico** usando el `defaultLang`.

</Steps>

## Qué Se Genera

Después de ejecutar `build`, tendrás:

- **CV PDF** - Generado desde HTML para cada idioma
- **LinkedIn markdown** - Perfil formateado para LinkedIn
- **GitHub Profile README** - Tu `README.md` para perfil de GitHub
- **Academics profile** - Credenciales académicas en markdown

<Aside type="tip">Revisa tu `package.json` para el campo `devsync.pathToCompiledCV` para ver dónde se genera tu CV HTML.</Aside>
