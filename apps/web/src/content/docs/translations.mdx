---
title: Translations
description: Available languages and their locale codes.
---

Devsync supports the following languages for UI labels in generated outputs.

| Code | Language | Locale |
|------|----------|--------|
| `en` | English | `en_US` |
| `es` | Español | `es_ES` |
| `fr` | Français | `fr_FR` |
| `pt` | Português | `pt_PT` |
| `de` | Deutsch | `de_DE` |
| `zh` | 中文 | `zh_CN` |
| `ja` | 日本語 | `ja_JP` |
| `ko` | 한국어 | `ko_KR` |
